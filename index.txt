Quelques exemples de balises en ligne et d'entités HTML

Le 1er exemple de balise en ligne que je vous donne est la balise sup. On l'utilise pour mettre une partie du texte en exposant. J'utilise aussi dans ce paragraphe la balise strong pour mettre très en évidence le mot " balise " et la balise code, expliquée au paragraphe suivant. Le mot " balise " est mis entre guillemets grâce à une entité HTML, le quote (voir 3ème paragraphe).

On utilise la balise code pour désigner une courte partie d'un code dans un texte. Ne surtout pas utiliser la balise code pour faire apparaître une partie de texte dans la police courrier, ce n'est pas du tout à ça qu'elle sert !! Sa seule utilisation correcte et autorisée est pour isoler du texte qui est vraiment du code et c'est tout !

Les entités HTML sont des codes qu'on utilise pour écrire en HTML certains caratères spéciaux (comme le & commercial, les guillemets, les symboles < ou >). Une entité commence toujours par le symbole & et se termine toujours par un ;. Par exemple " &amp; " est une entité HTML que l'on utilise dans le code HTML pour faire apparaître dans le navigateur le symbole &. Un autre exemple d'entité HTML est l'entité " &quot; " qui insère un guillemet.

La balise abbr, qui signifie " Abbreviation " est soulignée d'un trait pointillé. Elle est utilisée pour marquer une abbréviation ou un acronyme et indiquer son sens, via son attribut title, comme, par exemple, HTML.

De manière plus générale, tout caractère spécial peut être encodé sous forme d'entité, comme, par exemple, le "é" (e accent aigu), dont l'entité HTML est " &eacute; ". Une liste complète des entités HTML peut être consultée à l'adresse suivante : https://dev.w3.org/html5/html-author/charref.

Pour conclure, essayons d'utiliser les entités HTML pour la phrase suivante : Dès Noël où un zéphyr haï me vêt de glaçons würmiens je dîne d’exquis rôtis de bœuf au kir à l’aÿ d’âge mûr & cætera !
